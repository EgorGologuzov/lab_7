# Лабораторная работа 7 - Отчет

Выполнил: Гологузов Егор, группа Т-423901-ИСТ

### Итоги работы

Была реализована функция сканирования QR-кодов и 4 случая обработки содержимого:

1. Текст "дом" / "house": открытие активности с картинкой и описанием дома
2. Текст "машина" / "car": открытие активности с картинкой и описанием машины
3. Текст "дерево" / "tree": открытие активности с картинкой и описанием дерева
4. Другой текст: открытие активности с распознанным текстом и картинкой со знаком вопроса
